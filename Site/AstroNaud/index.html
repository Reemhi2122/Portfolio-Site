AstroNaud

I have made this game in Unity3D using Visual Studio C#. This game is a jumping game with a little twist. You have to climb on platforms and get as high as possible while avoiding enemys. But to get the actual points you need to return to the home ship at the bottom. This game is made for a arcade machine

This is a simple movement script for the game. It had to work with the animations and to make the player turn. also jumping needed to be perfect so it wouldn't be annoying. but you shouldn't be able to double jump.

void Update()
{
    float horizontal = Input.GetAxis("Horizontal");

    transform.Translate(horizontal * speed * Time.deltaTime, 0f, 0f);
    
    animator.SetBool("Iswalking", (Input.GetKey(KeyCode.A) || Input.GetKey(KeyCode.D)));

    mySrpiterenderer.flipX = (horizontal < 0);

    if (rb.velocity.y <= 0.05f && rb.velocity.y >= -0.05f)
        if (Input.GetButtonDown("Jump"))
            rb.AddForce(new Vector3(0f, jumpSpeed, 0f), ForceMode2D.Impulse);

    if (transform.position.x < -9)
        transform.position = new Vector3(9, transform.position.y, transform.position.z);

    if (transform.position.x > 9)
        transform.position = new Vector3(-9, transform.position.y, transform.position.z);

    if (health == 0)
        SceneManager.LoadScene("GameOverScreen");
}

After the movement was done. I made a random spawner for the tiles. I had a hard time making this correct because I had to make sure the actual player could reach every tile. The code I show you are for the power-up tiles. This function gets called everytime the player gets near the end. It calculates if it should make a normal, powerup or enemy tile.

void Spawner()
{
    for (int i = 100; i > 0; i--)
    {
        int special = Random.Range(1, 100);
        if (special <= 10)
        {
            GameObject cloneP = Instantiate(platformP, new Vector3(Random.Range(-7.21f, 7.16f), spawnP.position.y, 0), Quaternion.identity);
            cloneP.name = "Tile" + tileNumber;
            tileNumber++;
            float posP = cloneP.transform.position.x;
            if (posP > 4.5f || posP < -4.5f)
            {
                GameObject cloneX = Instantiate(platform, new Vector3(Random.Range(-1f, 1f), spawnP.position.y, 0), Quaternion.identity);
                cloneX.name = "Tile" + tileNumber;
                transform.Translate(0f, 3f, 0f);
            }
            else
                transform.Translate(0f, 3f, 0f);
        }
    }
}

The background is next to be spawned behind it. What I did was adding an array of game objects that I filled with 5 different backgrounds. In the for loop, it takes one of those random and places it. renames the background to order and places the next one 10 units higher. This was not that hard but I learned a lot from this back then.

void FixedUpdate()
{
    if (Camera.main.transform.position.y > distance)
    {
        SpawnerBG();
        distance += 294;
    }
}

public void SpawnerBG()
{
    for (int i = 30; i > 0; i--)
    {
        bgNumber++;
        GameObject clone = Instantiate(bg[Random.Range(1, 5)], new Vector3(transform.position.x, transform.position.y + thispos, transform.position.z), Quaternion.identity);
        clone.name = "bg" + bgNumber;
        thispos += 10;
    }
}
